# Bash Automation Toolkit (16 Scripts)

A curated collection of Bash scripts focused on **automation** for system administration, networking, security, and log analysis. [page:1]  
These scripts are designed as a mini-lab to showcase practical Linux skills for DevOps, SysAdmin, SOC, and cybersecurity roles. [page:1]

## Features

- System monitoring and health checks  
- Package installation and automation  
- Networking and basic security tooling  
- Log compression, parsing, and analysis  
- Utility scripts and scripting logic mini-projects
